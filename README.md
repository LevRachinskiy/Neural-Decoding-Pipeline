# Real-Time Neural Decoding Pipeline

## Overview
Built a real-time neural signal processing pipeline for streaming neural data. The system processes signals and extracts features for downstream machine learning models.

## Pipeline
1. Signal acquisition  
2. Bandpass filtering + preprocessing  
3. Feature extraction  
4. ML decoding  

## Structure
- src/pipeline.py: main entry point  
- src/acquisition.py: data ingestion  
- src/processing.py: signal processing  
- src/ml_node.py: model logic
